# Template Recomeço Gentil

Este template é para quando a pessoa sumiu, travou ou perdeu o ritmo.

## Para quem é

- Quem teve uma semana difícil
- Quem parou de abrir o app
- Quem sente vergonha de recomeçar
- Quem precisa voltar sem cobrança agressiva

## Como funciona

O template cria um reset de 3 dias.

### Dia 1

Só registrar como está.

### Dia 2

Escolher uma tarefa pequena.

### Dia 3

Voltar para o plano com ajuste.

## Regra principal

Recomeçar pequeno ainda é recomeçar.

## Resultado esperado

A pessoa volta sem sentir que perdeu tudo.

## Agente principal

A agente principal é a **Calma**.
