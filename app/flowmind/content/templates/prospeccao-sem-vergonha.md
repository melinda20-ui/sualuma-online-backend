# Template Prospecção Sem Vergonha

Este template é para quem trava na hora de vender.

## Para quem é

- Quem tem vergonha de chamar clientes
- Quem posta, cria, organiza, mas não vende
- Quem precisa de scripts simples
- Quem precisa acompanhar follow-ups

## Como funciona

O sistema cria uma rotina leve de conversas.

Inclui:

- Lista de contatos
- Script inicial
- Script de follow-up
- Script de reativação
- Registro de resposta
- Métrica semanal de conversas

## Regra principal

Venda começa com conversa, não com perfeição.

## Resultado esperado

A pessoa prospecta de forma mais humana e menos travada.

## Agente principal

A agente principal é a **Rica**.
