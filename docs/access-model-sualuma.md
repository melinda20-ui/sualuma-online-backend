# Modelo oficial de acessos Sualuma

## Admin
- Entra direto no Studio-lab.
- Domínio: https://studio.sualuma.online
- Pode acessar tudo.

## Cliente comum / contratante de serviços
- Dashboard: https://dashboardcliente.sualuma.online/member-services
- Acesso gratuito.
- Pode contratar prestadores.
- Comunidade liberada.
- Não tem agentes IA pagos, salvo extras comprados.

## Cliente IA
- Dashboard: https://dashboardcliente.sualuma.online/member-user
- Acessa dashboard de IA.
- Tem benefícios extras ao contratar prestador.
- Não vira prestador automaticamente.

## Prestador de serviço
- Dashboard: https://meuservico.sualuma.online/provider-services
- Acessa painel de prestação de serviços.
- Só pode divulgar/prestar serviço dentro do sistema.
- Não vira cliente IA automaticamente.

## Usuário com IA + Prestador
- Vai para portal de escolha.
- Mostra:
  - Entrar como Cliente IA
  - Entrar como Prestador

## Comunidade
- Todos entram.
- Bloqueia WhatsApp, serviço externo, venda por fora e divulgação sem plano de prestador.
